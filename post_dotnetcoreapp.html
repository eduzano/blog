Console Application with .NET Core

Let's print something with .NET Core!

  1. Open a new Powershell Console Prompt.

  2. Run the following commands.

    MKDIR -> Creates a new folder.

    CD -> Goes into the folder.

    DOTNET NEW CONSOLE -O -> Creates a new console application in the folder.

  3. Get into the folder and run the app!

My app prints "Hello SouthWorks" because i modified the Program.cs class with my custom message!
