MVC App with VS - .NET Core - Docker

In this part we are going to create a Model-View-Controller app with .NET Core, putting it into a container for future deploys in other platforms.

  1. Open a Visual Studio and create a New Project.


    1. Choose .NET Core.
    2. Choose "Web Application ASP.NET Core".
    3. Give a name to the project.

    1. Choose Web Application.

    2. Check Docker compatibility

    3. For this example we won't use authentication, but there you can choose between distinct auth types!

    4. In OS, you can choose what OS Docker will support in the container. For this example is Windows.

  4. Click accept and create the project.

  5. Click Play, this will compile and run your MVC app in the docker container.

  6. After clicking run, Visual Studio will compile the solution and open a new browser tab with a page.

    That page is the default web of the MVC project that we created, you will find there links to tutorials, news, etc.

    If we check the containers that our Docker have..

Following this steps made you have your own MVC application inside a container of Docker, you can export this image, import in another Docker, modify with your own code, create a new one, etc. Thanks for reading this post, hope it helps you!
